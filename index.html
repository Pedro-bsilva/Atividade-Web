ATIVIDADE DE PORTFÓLIO PARA AULA DE Desenvolvimento WEB.

Nesta atividade, proposta pelo professor Diogo,
estamos desenvolvendo um portfólio pessoal
utilizando apenas HTML e CSS. O objetivo é
aprofundar nossos conhecimentos nessas tecnologias
fundamentais para a criação de sites.
Este portfólio será uma vitrine das minhas habilidades,
experiências e projetos, refletindo o que aprendi
ao longo do curso de Análise e Desenvolvimento de Sistemas.
